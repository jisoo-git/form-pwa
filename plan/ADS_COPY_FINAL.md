## 구글 광고

### 1순위 — 구글 검색광고
> 구글에서 검색하는 학부모 타겟. 안산 외 서울·경기까지 커버.

**집행 키워드**
```
디미고 학원 / 디미고 입시 학원 / 디지털미디어고 학원 / 디미고 코딩 학원 / 디미고 입시 준비
```

**소재 1 — 실적형**

| | |
|--|--|
| 헤드라인1 | 디미고 합격률 전국 1위 |

| 헤드라인3 | 코딩·자소서·면접 완전 대비 |

| 설명2 | 디미고 동문 강사 + 30년 입시 전문가. 7월 개강 – 지금 바로 수강 신청. |

**소재 2 — 상담 유도형**

|--|--|

| 설명1 | 특별·일반전형 동시 대비 전략. 9년 기출·면접 DB로 합격률을 높이세요. |
| 설명2 | 궁금한 점은 카카오 채널로 바로 상담받으세요. 인코딩플러스. |

**타겟 설정**
```

```
