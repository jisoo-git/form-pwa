## 1. 광고 매체 기초 개념

| 이름 | 어디서 보이냐 | 어떻게 생겼냐 | 우선순위 |
|------|------------|------------|---------|
| **네이버 파워링크** | 네이버 검색 결과 맨 위 | "광고" 딱지 붙은 텍스트 링크 | ✅ 지금 |
| **네이버 파워컨텐츠** | 검색 결과 중간 | 블로그 글처럼 보이는 광고 | ✅ 지금 |
| **구글 검색광고** | 구글 검색 결과 맨 위 | "스폰서" 딱지 붙은 텍스트 링크 | ✅ 지금 |
| **네이버 브랜드검색** | "인코딩플러스" 검색 시 상단 전체 | 학원 정보 전체 노출 | 🔜 나중에 |
| **구글 디스플레이(GDN)** | 유튜브 영상 앞·중간, 각종 사이트 | 이미지·영상 배너 | 🔜 나중에 |
| **네이버 GFA** | 네이버 메인·뉴스·카페 지면 | 이미지 배너 | 🔜 나중에 |

**검색광고** = 이미 관심 있어서 검색한 사람한테 노출 (텍스트만)  
**배너광고** = 아직 검색 안 한 사람한테도 먼저 노출 (이미지 필수)

## 2. 학원 핵심 강점

| 강점 | 내용 |
|------|------|
| 합격 실적 | 9년 누적 212명, 2026년 37명 합격 |
| 전국 순위 | 디미고 합격률 전국 1위 |
| 전문성 | 디미고 4개 학과 모두 커버하는 유일한 전문 학원 |
| 강사진 | 디미고 동문 강사 + 30년 입시 전문가 |
| 통합 케어 | 코딩 실적물 → 자소서 → 면접까지 한 곳에서 완성 |

---

## 3. 학부모 심리 분석

- 디미고는 특수목적고 → 정보 부족 → **검증된 전문 학원** 의존도 높음
- 전형 복잡함 (특별·일반전형, 소질적성검사, 자소서, 면접) → **다 알아서 해주는 곳** 선호
- 자녀 입시 = 실패 허용 안 됨 → **숫자와 실적이 최고의 설득**
- 타겟: 학부모 1차 / 학생 2차 | 지역: 안산 기반 → 경기·서울 확장 가능

## 4. 카피 작성 원칙

**숫자를 쓰는 이유**
"좋은 학원입니다" 보다 "212명 합격"이 훨씬 믿음이 감. 숫자는 학부모한테 최고의 설득.

**소재를 2개씩 만드는 이유 (A/B 테스트)**
방향이 다른 광고 2개를 동시에 돌려서 어느 쪽 클릭이 더 많은지 비교 → 잘 되는 걸 남기고 안 되는 건 교체.
- 소재 1 (자랑형): 실적·숫자로 신뢰 먼저 줌 → "우리가 제일 잘해요"
- 소재 2 (경고형 or 행동유도형): 불안 자극 or 설명회 유도 → "지금 행동하세요"

**소재 2를 카카오 상담으로 유도하는 이유**
설명회는 연 2회뿐 → 날짜 기다리다 이탈 가능성 높음.
카카오 채널 상담은 상시 가능 → 광고 보자마자 바로 연결 가능.
학부모 입장에서 부담도 없고, 사이트에 버튼도 이미 있음.

## 5. 광고 소재

### 네이버 검색광고 (파워링크)

> 구조: 제목 15자 이내 + 설명 45자 이내

#### 소재 1 — 자랑형 (실적 강조)
> 네이버에서 "디미고 학원"을 검색한 학부모는 이미 찾는 상태. 숫자로 신뢰를 주는 게 가장 효과적.

```
제목: 디미고 합격률 전국 1위 인코딩플러스
설명: 9년 212명 합격. 4개 학과 전문. 코딩·자소서·면접까지 완성합니다.
```

#### 소재 2 — 상담 유도형 (카카오 채널 상담)
> 설명회는 연 2회뿐이라 날짜 기다리다 이탈 많음. 카카오 상담은 상시 가능 → 광고 보자마자 바로 연결.

```
제목: 디미고 입시 학원 무료 상담 인코딩플러스
설명: 2026 디미고 37명 합격. 궁금한 점 카카오 채널로 바로 상담받으세요.
```

---

### 구글 검색광고

> 구조: 헤드라인 3줄 (각 30자 이내) + 설명 2줄 (각 90자 이내)

#### 소재 1 — 자랑형 (실적 직격탄)
> 숫자와 실적으로 신뢰감을 먼저 줌. "전국 1위"가 첫인상에서 가장 강력.

```
헤드라인1: 디미고 합격률 전국 1위
헤드라인2: 9년 212명 합격 인코딩플러스
헤드라인3: 코딩·자소서·면접 완전 대비
설명1: 디미고 4개 학과 전문 학원. 소질적성검사부터 면접까지 한 번에 준비하세요.
설명2: 디미고 동문 강사 + 30년 입시 전문가. 7월 개강 – 지금 무료 설명회 신청.
```

#### 소재 2 — 상담 유도형 (카카오 채널 상담)
> 설명회는 연 2회뿐이라 날짜 기다리다 이탈 많음. 불안을 건드린 뒤 즉시 행동 가능한 카카오 상담으로 연결.

```
헤드라인1: 디미고 준비, 혼자 하면 위험합니다
헤드라인2: 전문가 무료 상담 바로 가능
헤드라인3: 안산 디미고 전문 입시 학원
설명1: 특별·일반전형 동시 대비 전략. 9년 기출·면접 DB로 합격률을 높이세요.
설명2: 궁금한 점은 카카오 채널로 바로 상담받으세요. 인코딩플러스.
```

## 6. 네이버 광고 전략 (매체별 키워드 + 소재)

### 1순위 — 파워링크
> 학원 찾으러 직접 검색하는 학부모 타겟. 가장 먼저 집행.

**키워드:**
```
디미고 학원
디미고 입시 학원
디지털미디어고 학원
안산 디미고 학원
디미고 코딩 학원
```
**소재:** 광고 카피 소재 1(실적형) + 소재 2(상담 유도형) 그대로 사용

---

### 2순위 — 파워컨텐츠
> 아직 학원을 찾는 건 아닌데 디미고 정보 검색 중인 학부모 타겟.  
> 블로그 글처럼 보여서 광고 거부감이 낮고, 정보성 콘텐츠로 신뢰를 먼저 쌓음.

**키워드:**
```
디미고 소질적성검사
디미고 특별전형 준비
디미고 면접 후기
디미고 자소서 쓰는 법
```
**소재:** 사이트 블로그에 이 키워드에 맞는 글 작성 → 파워컨텐츠로 등록  
(블로그 기능이 이미 있으니 글만 쓰면 됨)

---

### 3순위 — 브랜드검색
> "인코딩플러스" 검색 시 상단 전체를 학원 정보로 채움.  
> 브랜드 인지도가 쌓인 뒤에 집행. 지금 당장은 필요 없음.

**키워드:**
```
인코딩플러스
```

---

### 나중에 — 범위 넓힐 때 (디미고 외 특성화고)

```
소프트웨어고 학원
선린고 입시
특성화고 입시 학원
안산 코딩 학원
```

## 7. 구글 광고 전략 (매체별 키워드 + 소재)

### 1순위 — 구글 검색광고
> 구글에서 디미고 관련 검색하는 학부모 타겟.  
> 네이버보다 지역 제한이 적어서 안산 외 서울·경기 학부모까지 커버 가능.

**키워드:**
```
디미고 학원
디미고 입시 학원
디지털미디어고 학원
디미고 코딩 학원
디미고 입시 준비
```
**소재:** 광고 카피 소재 1(실적형) + 소재 2(상담 유도형) 그대로 사용

---

### 2순위 — 구글 디스플레이(GDN)
> 유튜브·뉴스·각종 사이트에서 이미지 배너로 노출.  
> 아직 검색 안 한 학부모한테 먼저 접근. 이미지 소재 제작 필요.  
> 브랜드 인지도 어느 정도 쌓인 뒤 집행.

**타겟 설정:**
```
지역: 안산 + 경기도 + 서울
관심사: 교육, 입시, 자녀 교육
```
**소재:** 이미지 배너 제작 필요 (지금은 보류)

---

### 네이버 vs 구글 차이점

| | 네이버 검색광고 | 구글 검색광고 |
|--|--------------|------------|
| **주 타겟** | 네이버 쓰는 학부모 (40~50대 多) | 구글 쓰는 학부모 (30~40대 多) |
| **지역** | 안산 중심 | 안산 + 서울·경기 확장 |
| **광고 형태** | 파워링크·파워컨텐츠 | 검색광고만 (디스플레이는 나중에) |
