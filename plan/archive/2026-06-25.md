## 오늘 한 작업

### 색상 교체 (2단계)

1. `#0ea5e9`(sky-500) → `#2563eb`(blue-600): 신뢰감 강화 목적
2. `#2563eb`(blue-600) → `#0099D6`(KA sky): 대한항공 리브랜딩 계열 적용

**최종 색상 토큰:**
```
Primary       #0099D6   버튼, accent bar, active 상태
Primary Dark  #0075A8   hover, 강조 텍스트
Primary Light #e0f4fb   배지 배경, 아이콘 bg
Primary Ring  #b3e5f5   선택 ring, step indicator
```

### 섹션 구분 방식 변경

- **이전**: 배경 `#fff` 단일 + 경계선 `1px solid #c8d0dc`
- **이후**: 배경 `#fff`↔`#fafafa` 교차 + 경계선 제거 + 패딩 52px
- **이유**: 섹션이 짧아서 경계선이 촘촘하게 보여 정신없어 보임

### 홈 배너 개선

- 연한 파스텔 배경 → 다크 그라디언트 (배너1: KA navy→sky)
- 텍스트: 어두운 색 → 흰 텍스트
- 화살표 버튼: 불투명 흰색 → 글래스(반투명+blur)
- `image?` 필드 추가 → URL 넣으면 사진 배경 자동 적용

### 기타

- TopNav 경계선: `#e3e8ee` → `#c8d0dc`
- 태블릿 레이아웃: UserLayout `max-w-[440px]` 제거
- BottomNav: maxWidth 제거
- "TRACK RECORD" → "합격 실적"
- 강좌 카드 클릭 → `/courses` 이동
- CTA: 버튼·전화 세로 분리, 학원소개·사업자번호 복원
- 비표준 색상(`#f0f9ff`, `#c4c4c8` 등) 정리
