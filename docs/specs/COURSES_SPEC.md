## COURSE 01 — 입시 단기특강

| 항목 | 내용 |
|------|------|
| 전형 | 특별전형 + 일반전형 (모두) |
| sub | 특별전형과 일반전형 모두 도전 |
| 수강료 | 73만원 / 월 · 4회 기준 |

### 상세 설명
두 전형을 모두 도전하여 합격률을 높이는 전략입니다.

특별전형 대비부터 일반전형 대비까지 16주에 걸쳐 종합적으로 대비합니다. 2027학년도 과정은 7월 18일에 개강합니다.

### 수업 구성

| 영역 | 내용 |
|------|------|
| 코딩/콘텐츠 | 실적물 생산, 이론 교육 |
| 적성고사 | 이산수학, 정보학습, 실전모의고사 |
| 입시논술 | 실적설명서, 자소서, 면접대비 |

### 수업 시간

| 요일 | 시간 | 형태 |
|------|------|------|
| 매주 토요일 | 12:00 – 18:00 | 대면 6시간 |
| 매주 수요일 | 오후 10시 또는 11시 | 비대면 1시간 · 보충 불가 |

## COURSE 02 — 일반전형 특강

| 항목 | 내용 |
|------|------|
| 전형 | 일반전형 |
| sub | 일반전형 집중 전략 |
| 수강료 | 48만원 / 월 · 4회 기준 |

### 상세 설명
일반전형은 내신과 소질적성검사가 핵심입니다. 기본 정보 역량과 적성고사 대비 정도에 따라 합격이 좌우됩니다.

코딩 수학과 정보 교육을 중심으로 소질적성검사를 대비하고, 면접을 함께 준비합니다.

### 수업 구성

| 영역 | 내용 |
|------|------|
| 적성(수학) | 이산수학을 바탕으로 수학 수업 진행 |
| 정보(컴퓨터) | 컴퓨터 교과 내용과 상식 차원의 정보 수업이 진행 |
| 입시논술 | 인성 면접 대비 |

### 수업 시간

| 요일 | 시간 | 형태 |
|------|------|------|
| 토 또는 일요일 | 15:00 – 18:00 | 대면 3시간 |
| 매주 수요일 | 오후 10시 또는 11시 | 비대면 1시간 · 보충 불가 |

## 수강신청 폼 구성 가이드 (FormBuilder 설정)

> Apply.tsx는 FormBuilder 활성 폼의 **섹션 = 스텝** 구조로 완전 동작.
> 아래 구성으로 폼을 만들어 isActive 활성화하면 수강신청이 열림.

### 섹션 1: 신청 확인

| 타입 | 레이블 | 필수 |
|------|--------|------|
| info | 수강신청 시 입력하신 개인정보(이름, 연락처, 학교 등)는 수업 운영 및 안내 목적으로만 사용되며, 제3자에게 제공되지 않습니다. | — |
| radio | 개인정보 활용에 동의하십니까? (옵션: 네 / 아니오) | ✓ |
| radio | 원하시는 수업을 선택해주세요 (옵션: 입시 단기특강 / 일반전형 특강) | ✓ |

> **radio '아니오' 선택 시** → 경고 메시지 표시 + 다음 버튼 비활성화  
> **수업 선택 레이블**에 반드시 "수업" 단어 포함 → 제출 시 `course` 필드 자동 추출

### 섹션 2: 유의사항

| 타입 | 레이블 내용 | 필수 |
|------|------------|------|
| info | 수업 개설 기준: 수업은 최소 2명 이상 등록 시 개설됩니다. 인원 미달 시 개별 안내 후 환불 처리됩니다. | — |
| info | 결석·보충 안내: 수요일 비대면 수업은 보충이 불가합니다. 대면 수업은 1회에 한해 사전 연락 후 보충 가능합니다. | — |
| info | 수강료 납부: 수강료는 매월 첫 수업 전까지 납부해 주세요. 미납 시 수업 참여가 제한될 수 있습니다. | — |
| info | 환불 규정: 수강 취소는 개강 1주일 전까지 전액 환불 가능합니다. 이후 취소 시 이미 진행된 수업 횟수에 따라 차등 환불됩니다. | — |
| checkbox | 위 유의사항을 모두 확인했습니다 (옵션 1개) | ✓ |

> **info 타입** → 파란 배경 공지 카드로 렌더 (질문 번호 없음)

### 섹션 3: 정보 입력

| 타입 | 레이블 | 필수 |
|------|--------|------|
| short | 이름 | ✓ |
| short | 학교명 | ✓ |
| radio | 학년 (옵션: 중3) | ✓ |
| short | 연락처 (학부모) | ✓ |
| long | 기타 문의사항 | — |

> **레이블 키워드** → 제출 시 자동 추출:  
> `이름` → name, `학교` → school, `연락처`/`전화` → phone
